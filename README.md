# Nuho Living Interaction Reference

This project is created as a reference for interaction design implementation.

## Purpose
To showcase how interaction, layout, and behavior should work on the landing page.

## Scope
- Homepage
- About Us

## Notes
This is not production code.  
This project is intended as a UI/UX and interaction reference for developers.

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Getting Started

Run the development server:

npm install  
npm run dev  

Open http://localhost:3000

## Key Focus
- Interaction behavior (hover, transition, scroll)
- Layout structure
- Visual consistency between sections

## Usage
Use this project as a reference when implementing the actual product in a separate codebase.
